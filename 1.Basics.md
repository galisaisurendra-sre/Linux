## What is an Operating system

```
An OS is software that acts as a middleman or bridge between computer Hardware and the computer user. It provide a user interface and the controls the computer hardware so that software can function.

Typer of OS:

1. Desktop Operating Systems - Microsoft Windows, macOS and Linux such as Ubuntu.
2. Server Operating Systems - Windows Server , Linux distributions like CentOs, Red Hat Enterprise Linux.
3. Mobile Operating Systems - Android, iOS, Windows Mobile.
4. Embedded Operating Systems - used in devices like router, smart TV, automobiles, home appliances.
5. Real-Time Operating Systems - used in medical equipment, car ECU, aerospace, defense, network firewalls, home security systems.

```
